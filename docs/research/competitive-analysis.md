# 11408study 竞品调研报告

## 摘要

本报告对 8 款与 11408study 目标人群相近的中外学习产品做了功能对照：粉笔考研（垂域大模型 + AI 老师）、知能行考研数学（自适应推题 + 知识点等级）、考研帮（一站式社区 + 题库）、小猿 AI（拍照解题 + DeepSeek 讲解）、StudyX（PDF→闪卡→测验闭环）、Quizlet（AI 闪卡 + Q-Chat 导师）、Anki（SM-2/FSRS 间隔复习生态）、多邻国/扇贝（游戏化打卡 + 同伴助学）。本仓已具备知识图谱、SM-2、拓扑路径、测验、笔记、AI 抽取与认证；显著缺失的是"AI 讲题与答疑、能力等级量化、自适应推题、错题本闭环、游戏化打卡、视频/PDF 时间戳定位"。文末按"价值×成本"给出 5 个候选新功能。

## 平台快览

### 1. 粉笔考研（fenbi.com）
- **核心功能**：垂域大模型驱动的 AI 老师（学情分析/规划/答疑/督学/解压 7 角色）；AI 申论批改多维结构化反馈；靶向练习（仅推送薄弱领域题目）；动态学习路径千人千面。
- **借鉴度**：高 | **复杂度**：L（垂域模型 + 7×24 答疑链路）

### 2. 知能行考研数学（bestzixue.com）
- **核心功能**：知识点拆到最小粒度并打 1–5 级"能力等级"；30 分钟初测画弱点画像；艾宾浩斯曲线在页面上以"小黄点"提醒复习；新题与错题混合推送；进度量化到小时。
- **借鉴度**：高 | **复杂度**：M（算法逻辑清晰，可在已有图谱上叠加）

### 3. 考研帮 App（kaoyan.com，好未来）
- **核心功能**：5 万+套真题在线刷；单词闪背；倒计时日历；AI 择校；论坛社群；自习室（番茄钟+在线人数氛围）。
- **借鉴度**：中 | **复杂度**：S–M

### 4. 小猿 AI（猿辅导）
- **核心功能**：OCR 拍照搜题（90% 准确率）；接入 DeepSeek 满血推理做"讲题"而非"给答案"；自动错题本 + 相似题推荐；学习数据看板。
- **借鉴度**：高 | **复杂度**：M（OCR 走第三方 API + LLM 讲解 prompt）

### 5. StudyX
- **核心功能**：PDF/文字一键生成笔记摘要；笔记 → 闪卡 → 测验题三步链路；YouTube 视频转文字摘要；分步解题。
- **借鉴度**：高 | **复杂度**：S（与本仓 AI 抽取链路高度兼容）

### 6. Quizlet（含 2026 新版）
- **核心功能**：AI 闪卡生成器（文本/文档→卡组）；Q-Chat 自适应 AI 导师；Coconote 音视频转笔记+卡+测验+播客；Learn 模式带间隔重复。
- **借鉴度**：高 | **复杂度**：M

### 7. Anki / FSRS 生态
- **核心功能**：SM-2 / FSRS-5 算法；卡组共享市场（27 考研 408 卡组 2190 张）；多端同步；自定义卡片模板（Cloze、双面）；统计图表（保留率曲线）。
- **借鉴度**：中（本仓已有 SM-2，可对标卡组分享与 FSRS-5）| **复杂度**：S–M

### 8. 多邻国 + 扇贝单词
- **核心功能**：闯关式课程树（多邻国）；红心血条与每日任务；连续打卡 streak；同伴助学（互相提醒、表情激励、断签即解约）；徽章/排行榜；猫头鹰 IP 化督学。
- **借鉴度**：中 | **复杂度**：S（前端为主）

## 对 11408study 的功能差距清单（Top 5，按"价值×成本"排序）

### 1. AI 讲题 + 错题本闭环（价值高 / 成本 M）
**场景**：用户在 Quiz 答错后点"问 AI"，弹出对话框，AI 基于题干、用户错选、关联知识点节点给出讲解；错题自动入"错题本"，按知识点聚合，并在下次复习时优先推送相似题。
**后端**：扩 `QuizService` 增 `wrong_answers` 表与聚合查询；`AiClientService` 加 `explainQuestion(questionId, userAnswer)` 方法，prompt 注入题目+图谱节点上下文。
**前端**：题目结果页加"AI 讲解"抽屉；新增"错题本"路由，按知识点分组，支持一键加入复习队列。

### 2. 知识点能力等级（1–5 级）+ 自适应推题（价值高 / 成本 M）
**场景**：每个图谱节点显示 1–5 级进度环；系统按答题正确率、连续正确次数动态升降级；学习页"今日推荐"按等级 + 遗忘风险智能挑题，而非固定拓扑序。
**后端**：新增 `user_topic_mastery(user_id, topic_id, level, score, updated_at)`；在 `StudyPathService` 旁加 `AdaptiveRecommender`，融合 mastery 分与 SM-2 due 时间。
**前端**：图谱节点叠加进度环 SVG；首页增"今日靶向题"卡片。

### 3. 资料 PDF 时间戳/页码定位 + 视频笔记联动（价值高 / 成本 M）
**场景**：导入王道讲义 PDF 或 B 站视频后，AI 抽取的知识点带"出处"（页码 / 时间戳）；点击知识点可直接跳到 PDF 对应页或视频对应秒数。
**后端**：扩 import pipeline 在 LLM prompt 里要求返回 `source_locator`；存到 `topic_source(topic_id, material_id, page, ts_seconds)`。
**前端**：PDF 用 `react-pdf` + 锚点；视频嵌 B 站 iframe 用 `seek` 接口；知识点卡片加"查看出处"按钮。

### 4. 游戏化打卡：streak + 每日任务 + 徽章（价值中 / 成本 S）
**场景**：登录首页显示连续打卡天数（火苗图标）；今日任务三件套（5 道新题/10 道复习/1 篇笔记）；满 7/30/100 天发徽章；断签提示挽留。
**后端**：`user_daily_task(user_id, date, new_quiz, review_quiz, note_count, completed)` + cron 每日重置；徽章规则配置化。
**前端**：首页 streak 火苗动画；任务进度条；徽章墙页面。

### 5. 番茄钟自习室 + 学习数据周报（价值中 / 成本 S）
**场景**：右上角浮动番茄钟（25/5），与"在线自习人数"显示制造氛围；每周日推送本周学习数据（专注分钟、完成题数、知识点新增、易错点 Top 3）。
**后端**：`focus_session(user_id, start, end, type)`；现有 `StatsService` 加周报聚合 + 邮件/站内信。
**前端**：复用 shadcn Sheet 做浮动番茄钟；周报页用现有图表组件即可。
